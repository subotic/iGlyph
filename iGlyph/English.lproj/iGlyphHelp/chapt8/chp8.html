8. Help

Use the menu <Help> (Fig.8.1) to show the different types of help.

{bmc bm76.WMF}
Fig.8.1

1. Help (VisualGlyph Help): Here you can call up help in the form of a window help file (extension .hlp). The help window will always stay on top of all other windows if the checkbox <Help Window always on top> is checked (see 2.5).

2. Tips: By default you will see the <Tips OnMouseMove> window. If you move the cursor over an object it will show either a short yellow <Tooltip> text or this window. It displays the functions you can perform with the keyboard or with the mouse buttons. Fig.8.2 is an example of the help you get when the mouse cursor moves over the writing area.

{bmc bm77.WMF}
Fig.8.2

3. Magnifier: Click on this menu item to open the magnifier window, which shows the contents of the writing area at the cursor position with a scale factor of 1:2. The function is active only if the VisualGlyph main window is maximized.
