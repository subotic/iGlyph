9. File Functions

Most of the programs working with files offer basic file functions in a <File> menu. VisualGlyph has the same menu (Fig.9.1):

- New (Create a new file)
- Open (Open an existing file)
- Save (Save a currently open file)
- Save as (Save a currently open file under a different name)
- Quit (Close the program)
- Example.IG (this is the name of the previously loaded file. A click on this line reloads that file)

{bmc bm79.WMF}
Fig.9.1

In order to open a IG-document that already exists, click on the menu item <Open>. The window standard dialog appears and you can select a file. VisualGlyph creates Files with the extension <.IG>.

{bmc bm80.WMF}
Fig.9.2: The Open Dialog (Win 2000)


The <Print> Dialog has three options:

- Print all pages of the document
- Print from page x to page y
- Print the current page
- Print multiselected objects

At the bottom of the print-dialog you can see the currently selected printer. By clicking on <Select Printer> you can select a different printer (Fig.9.3b).
