Appendix C: Program Bugs

1. Rotated Hieroglyphs (Chapter 4.4.3): Rotated hieroglyphs are transferred correctly from VisualGlyph to MS-WORD. If the user DoubleClicks on the WMF-Graphic the Word Graphic-Editor loses the rotation angles and other properties might change too.
Comment: This is probably a mistake or limitation of the WORD Editor.

2.Edit Text Dialog (Chapter 5.2): At the right-hand side of the <Edit Text> dialog you see a table with all signs of the font. Here the signs are drawn like pictures. In this case all signs of a font can be shown. If the user clicks on a sign to transfer it to the textbox then some signs (with an ASC value greater then 128) are not transferred.
Comment: This is probably a mistake or limitation of the textbox-control.

3. Dashed Lines: A dashed line is created with the system function dotted line (see chapter 5.5). Within VisualGlyph this is of no importance, and if the user copies such a line to MS-WORD then it is shown as dotted.
Comment: The appearance of the dotted or dashed line is caused by an unknown effect of the system function.

4. Hieroglyphs and Text do not appear on the screen: This error may appear in many different ways. One case will be described here:

{bmc bm83.WMF}
Fig. C.1

Fig. C.1 shows the page format A3, but only a part of it is visible on the screen, here colored as yellow. The text "New Text" was printed on the right page. The user sees the word "New", by scrolling to the right he will normally see the full text. But on one computer the word "Text" was not shown. The phenomena occured with font signs in general but not with line drawings and depends on the graphics card (in this case NVidia Geforce MX 2000 was installed). Reducing the max. speed corrected this error. Another work around was to turn <font smoothing> on (Menu: View/Font Smoothing), but this only had an effect when setting the font size to a larger scale.

In all probability video drivers can cause this type of errors, so that you should try to install the correct driver and its latest version.

5. Exported Hieroglyphs are cut: It might occur that signs are cut in WORD or other programs. In this case the user can manually set the surrounding blue frame in Visualglyph by pressing Cntrl-LeftArrow, Control-RightArrow, Control-UpArrow, Control-DownArrow. After this you can repeat the export action (Cntrl-W). For further information see 7.2. Export Data to Word Processors etc.
