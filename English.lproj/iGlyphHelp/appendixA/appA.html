Appendix A: Key Strokes

1. Keys

1. General

Control-A Append a page at the end of the document
Control-C Copy to clipboard
Control-D Delete the current page
Control-F Fast drag and drop
Control-I Insert a page before the current page
Control-N Create a new document
Control-O Open an existing document
Control-P Call the print dialogbox
Control-Q Quit the program
Control-R Open the dialogbox: program settings
Control-S Save the current document
Control-T Open the OnMouseMove tip window
Control-U Open the dialogbox: document settings
Control-V Paste clipboard contents
Control-W Copy as WMF (Word etc.) (MicroSoft format to exchange data, e.g. to MS-Word)
Control-X Cut data (data is copied to the clipboard)

Control-PageDown Go to the first page
PageDown Go to the previous page
Control-PageUp Go to the last page
PageUp Go to the next page

F1 Call help
Delete Delete (multi)selected objects
LeftArrow Move (multi)selected objects one pixel to the left
UpArrow Move (multi)selected objects one pixel up
RightArrow Move (multi)selected objects one pixel to the right
DownArrow Move (multi)selected objects one pixel down

2. Frame of multiselected hieroglyphs (see 7.2. Export Data to Word Processors etc.)

Control-LeftArrow Move left margin by one pixel to the left
Alt -RightArrow Move left margin by one pixel to the right

Control-RightArrow Move right margin by one pixel to the right
Alt-LeftArrow Move right margin by one pixel to the left

Control-UpArrow Move top margin by one pixel upwards
Alt -DownArrow Move top margin by one pixel downwards

Control-DownArrow Move bottom margin by one pixel downwards
Alt-UpArrow Move bottom margin by one pixel upwards




2. Keys and Mouse

1. Toolbox <Hieroglyphs> (see 4.2)

LeftMouseDown: Transfers a sign to the cursor position in the writing area
RightMouseDown: Changes the value of a selected sign
Alt+LeftMouseDown: Transfers a sign value to the <Group> field (see 4.3.1)


2. The Blue Handle (see 4.4.1)

RightMouseDown: Shows "Hieroglyph" Info (see 4.5)
Alt+LeftMouseDown: Transfers the sign value to the <Group> field (see 4.3.1)
